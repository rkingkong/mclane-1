CSV Image Uploader

This module helps to import images for Product Variants, Product Extra Images and Website Product Categories.

Basic Features
  • This module helps to import images for Product Variants, Product Extra Images and Website Product Categories using Zip file.
  • Allow user to modify existing product image.
  • Allow user to modify existing product Extra images or create extra images.
  • Allow user to modify existing website Categories image.

Workflow

Create Zip File

How we create Zip file:

  • First of all, You will create a folder named 'Product'.
  • Add images folder inside Product folder
  • Add CSV file inside the folder.
  • Compress that folder and make zip file.
